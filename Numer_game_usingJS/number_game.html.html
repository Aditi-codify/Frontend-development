Guess The Number(using HTML and JAVA SCRIPT)

Step 1 : THE prompt will ask you to enter a max number.

Step 2 : After which it will decide a number of its own within the max range you set.

Step 3 : The while loop in JS will run untill you enter(guess) the right random number or unless you enter quit.

Step 4 : You will be ending the game automatically when you enter the right guess.
